Firebase setup

You only need to do this once. It takes about 5 minutes and gives you a free real-time database shared between both phones.

Step 1

Go to console.firebase.google.com and sign in with your Google account.

Step 2

Click Add project → name it 10k-tracker → disable Google Analytics → Create project.

Step 3

In the left sidebar go to Build → Realtime Database → Create database → choose your region → Start in test mode → Enable.

Step 4

Go to Project settings (gear icon top-left) → scroll to "Your apps" → click the Web icon </> → register app as 10k-tracker → copy the firebaseConfig object that appears.

Step 5

Open index.html in a text editor and replace the placeholder values in the firebaseConfig block near the top of the file with your real values. Save and re-deploy to Netlify.

The config looks like: apiKey, authDomain, databaseURL, projectId, storageBucket, messagingSenderId, appId.
